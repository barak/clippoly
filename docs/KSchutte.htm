Research

In Februari 1994 I got my PhD on the thesis "Knowledge Based Recognition of Man-Made Objects". An abstract in Dutch is available, as well as a shorter abstract in English, but not the complete thesis, due to disk space (and network bandwidth) reasons. Please contact me if you are interested in my thesis.

My current research project is the model-based analysis of sequences of images. Spatial and modality sequences of scanned maps are under study.
Currently, I am working on papers on:

  • Mosaicking of scanned maps
  • Recognition of buildings from multiple aerial images (to be submitted: Real Soon Now)
  • Improving segmentation by using shape knowledge
  • An interactive tool for 3D scene analysis (submitted: june 1995 to CVIP)
  • Polygon clipping uuencoded version (submitted 7 july 1995 to ACM ToG)

A paper on mosaicking of scanned maps (html version) (PostScript version) uuencoded PostScript version is available, as well as a html and a PostScript version of a paper on generating object hypotheses from multiple image.

A publication list is available.

Available Software

I have made some software available. Feedback on the code is appreciated.

A note on downloading: currently I have no ftp directory to store the files on, so they are available by http only. This means that you explicitely have to select a download option on your WWW browser; in general, clicking and saving the file does NOT work.
Special notice for Windows 95 users: the Netscape 2.01 GOLD browser for Windows 95 seems not ignore the .gz suffix for .tar.gz files. This will lead to errors for using tar only. You >>have<< to use gunzip to decompress the files!
For all those files an uuencoded version also has been made available. Those files do not contain binary characters and thus should be downloaded without any problems.

Clippoly


Clippoly is an algorithm to clip two polygons. A description of the algorithm is available as well as some comments about the code. On 7 July 1995 I have submitted a paper uuencoded version on this algorithm to ACM Transactions on Graphics. (New news: this paper is not accepted by ToG -- they thought it was too similar to Weiler-Atherton.) The gzipped tar file (patchlevel 6) uuencoded version contains the source code, but not all the descriptions mentioned above.
